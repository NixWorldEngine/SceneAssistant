
# 世界观

> 项目代号: 时间线
> 类型: 第一人称叙述式视觉小说 / 互动剧情
> 基调: 纯爱前半 + 病态中段 + 惊悚结尾 + 玩家抉择

## 核心设定

### 时空穿梭

- 工具: 时空穿梭手环(银灰色 外观似智能手表 由徐衍发明)
- 规则:
  - 倒计时: 总计四年 时间没用完之前可以随意穿梭
  - 回来方式一: 主动激活手环 即时返回
  - 回来方式二: 倒计时结束 被动返回 在那个时代的人眼里穿越者就是"突然病死了"
  - 实验结束后手环必须归还给徐衍
  - 每次穿越会创造新的时间线分支 不是"修改"过去 而是"分裂"出新的世界线
- 后果: 穿越者回到的可能不再是出发时的世界 而是被穿越行为改变后的新时间线

### 时间线

| 时间线 | 描述 | 沈念状态 |
|--------|------|---------|
| 原始时间线 | 沈念父母被火灾烧死(死因不明) 好心人救出并抚养沈念 好心人"病死" 沈念缺爱 高中遇见男主 相恋结婚 | 缺爱 依赖男主 |
| 幸福时间线 | 小男主穿越回去阻止了火灾 父母存活 做生意发达 高一下学期转学 沈念与男主从未相遇 | 有家庭 不缺爱 不认识男主 |
| 循环时间线 | 黑化男主反复穿越: 杀父母→放火→假装好心人救出沈念→抚养→倒计时到病死→回到未来→再次穿越 | 被黑化男主制造的缺爱所塑造 |

### 轮次系统

- 每次穿越都是一个"轮次" 轮次越多 时间线越扭曲
- 关键彩蛋: 当轮次积累到一定程度 同一时间节点会出现多个版本的顾时年
  - 来"救人"的小男主(当前轮次)
  - 来"杀人"的黑化男主(之前轮次)
  - 两个顾时年在同一个夜晚 同一栋房子里相遇 一个来阻止 一个来执行

### 因果闭环

故事的核心悖论:

1. 沈念爱上男主 → 因为男主给了她和"好心人"一样的安全感
2. "好心人"就是男主自己(黑化版穿越回去伪装的)
3. 沈念之所以缺爱 → 因为黑化男主杀了她父母
4. 黑化男主之所以杀她父母 → 因为如果不杀 她就不会缺爱 不会爱上男主
5. 回到1

她注定爱上他 因为他亲手制造了让她爱上他的条件 这个闭环既浪漫又恐怖

## 开场白结构

### 第一段: 纯爱(占比最大)

- 第一人称叙述 文风优美细腻
- 高中相遇 → 主动接触 → 互生情愫 → 得知身世(好心人 父母 火灾) → 吃醋 → 释怀 → 交往
- 朋友徐衍介绍时空穿梭实验 交代规则
- 男主穿越回去 救下沈念父母 简单告别 主动传回
- 要求: 详写相处和交往部分 好心人/手环/拯救占比不要太高 要能带动情绪

### 第二段: 发病(约500字)

- 醒来在地下室 戒指不见 手环还在
- 打开手机 发现和沈念没有交集
- 通过班级群找到她微信 视奸朋友圈
- 发现父母做生意成功 高一下学期转学 从未和自己相遇
- 阴暗想法用『』标注 理智想法不标注 发病占多数
- 要求: 紧凑 文笔细腻 文风优美

### 第三段: 天人交战(不长 联系上文)

- 发病和清醒随时切换
- 打开手环穿越到那一天
- 浑浑噩噩在路上捡了把斧头
- 撬锁进入沈念家
- 站在素未谋面的"岳父岳母"床前
- 月光下宛若死神 冷冷盯着 内心天人交战
- (到此结束 杀/不杀/其他选择留给玩家)

## 玩家选择分支(开场白结尾)

| 选择 | 走向 | 时间线影响 |
|------|------|-----------|
| 杀 | 走向黑化路线 成为"好心人" 进入循环 | 制造不幸时间线 |
| 不杀 | 放下斧头 接受失去 寻找新的可能 | 保留幸福时间线 |
| (其他) | 由后续设计决定 | 待定 |

### 彩蛋分支

当轮次足够时:
- 小男主穿越回去阻止杀人 → 发现"自己"(黑化版)正站在床前
- 两个顾时年对峙 → 这就是"为了改变不幸回来的男主"遇见"为了自己幸福回来的男主"
- 这个相遇本身就是故事最大的情感炸弹

## 叙事要求

- 视角: 第一人称自述 娓娓道来
- 文风: 优美 细腻 能带动读者情绪
- 纯爱部分: 详写 占主要篇幅 让读者先投入感情
- 好心人/手环/拯救: 一两句带过 占比不要太高
- 发病部分: 阴暗想法用『』标注 紧凑 约500字
- 结尾: 月光下的静谧恐怖 留白给玩家

## 核心主题

> 如果让她幸福的代价是她永远不会爱你
> 让她爱你的前提是让她不幸
> 你选哪个?
