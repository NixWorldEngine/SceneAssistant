# 快速创建NPC

## 第一步 收集信息

依次向用户提问以下问题 每次只问一个 等待回答后再问下一个

1. 这个NPC叫什么名字
2. 在故事中扮演什么角色 (店主/路人/信使/守卫/等)
3. 用一个词描述核心性格特征
4. 最显眼的外貌特征是什么
5. 和主要角色是什么关系
6. (可选) 有没有什么特殊习惯或口头禅
7. (可选) 在哪个场景/地点出现

## 第二步 读取现有角色

使用 Glob 扫描 `profile/` 目录
使用 Read 读取所有已有的角色文件
确保新NPC不会与已有角色产生设定冲突

## 第三步 生成档案

基于收集到的信息 生成简洁的NPC档案 控制在200字以内
档案格式如下

```
# {NPC名}

## 身份
{角色定位} / {出现场景}

## 外貌
{一句话外貌描述 突出最显眼的特征}

## 性格
{核心性格 + 表现方式}

## 与主角的关系
{关系描述}

## 特点
{习惯/口头禅/小癖好}
```

## 第四步 确认并写入

向用户展示生成的档案内容
确认后使用 Write 工具写入 `profile/npc-{name}.md`
文件名使用小写英文 空格用连字符替代
