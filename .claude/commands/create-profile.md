# 从参考资料生成人设档案

## 第一步 读取参考资料

使用 Glob 工具扫描 `reference/` 目录中的所有文件
使用 Read 工具逐一读取所有文件内容

## 第二步 分析与分类

分析所有参考资料 识别以下内容类型

- 角色相关的描述/灵感/片段 -> 将生成 `character-{name}.md`
- 世界观/地理/历史/体系 -> 将生成 `world-{aspect}.md`
- 阵营/势力/组织 -> 将生成 `faction-{name}.md`

向用户报告分析结果: 预计生成哪些文件 每个文件的大致内容

## 第三步 逐一生成

对每个待生成的文件 先向用户展示预览内容 等待确认后再写入

### 角色档案模板 character-{name}.md

```
# {角色名}

## 基本信息
- 姓名:
- 年龄:
- 性别:
- 外貌:

## 性格
- 核心特质:
- 处事方式:
- 情感模式:

## 背景
- 出身:
- 关键经历:
- 当前状态:

## 关系
- {与其他角色的关系}

## 说话风格
- 用词习惯:
- 语气特点:
- 口头禅:

## 其他
- 特殊能力:
- 弱点:
- 秘密:
```

### 世界观档案模板 world-{aspect}.md

```
# {世界观方面}

## 概述

## 核心规则

## 关键要素

## 与角色的关联
```

### 阵营档案模板 faction-{name}.md

```
# {阵营名}

## 概述

## 组织结构

## 核心理念

## 成员

## 与其他势力的关系
```

## 第四步 写入文件

用户确认后 使用 Write 工具将内容写入 `profile/` 目录
文件名使用小写英文 空格用连字符替代

## 第五步 完成报告

输出生成摘要: 共生成多少文件 涵盖多少角色/世界观/阵营
