## 通用技术规范（所有风格共享）

- 输出单个 `.html` 文件，内联所有 CSS (`<style>`) 和 JS (`<script>`)
- 必须包含 `<!DOCTYPE html>` 和 `<meta charset="UTF-8">`
- 响应式适配：手机 (`<768px`) 和桌面 (`>=768px`)
- 字体通过 Google Fonts `<link>` 引入，中文回退至 Noto Sans/Serif SC 或系统字体
- 图片使用占位符 (`https://picsum.photos/宽/高`) 或内联 SVG
- 图标使用内联 SVG 或 CSS/Unicode 符号，不依赖外部图标库
- 优先 CSS 动画，JS 仅用于粒子系统和交互逻辑
- 所有交互元素包含平滑过渡 (transition)
- `user-select: auto`，不要禁止用户选择文字
